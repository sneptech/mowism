# /mow:help

Show available MOW commands and usage guide.

## Usage

    /mow:help

## What Happens

1. Displays the complete MOW command reference
2. Lists all available /mow:* commands grouped by category
3. Shows brief descriptions for each command
4. No project-specific analysis or suggestions -- just the reference

## Examples

    /mow:help    Show all available MOW commands

## Related

    /mow:help-open <command>    Open detailed help for a specific command
    /mow:progress               Check project status and next steps
    /mow:settings               Configure MOW workflow preferences
